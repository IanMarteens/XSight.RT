The Options dialog box can be executed from the Project|Options menu command:

Ray tracer

The Ray tracer page contains options for fine tuning the scene optimizer:

OptionMeaning
Loop expansion Maximum size allowed for inlined repeaters.
Union threshold Minimum item count in unions for activating bounds checking.
Diffusion levels Number of levels where sampler will cast an additional ray for diffuse reflectors.
Bounding spheres Controls the expected performance of bounding spheres.

Sonar mode
