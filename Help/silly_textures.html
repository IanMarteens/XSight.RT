Small Instantiation Language: Materials

See also

Each shape in a XSight RT scene must be associated with a material. This material controls how the object surface reacts to the incoming light ray, the amount of diffused light and the object color, the intensity of the reflected ray, if any, and its own color.

Supported materials

XSight RT currently supports six classes of material:

  • Plastic
  • Metal
  • Marble
  • Spotted
  • Pigment
  • MetalPigment
  • Glass

They can be classified as in the following image:

The active render quality also controls how these materials are rendered.

Common material features

All these materials share some common features:

  • Perfect specular reflection
  • Diffuse specular reflection
  • Phong specular highlights
  • Normal vector perturbation
