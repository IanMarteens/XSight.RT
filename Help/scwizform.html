The Scene Wizard provides a simple but effective tool for creating new scene descriptions.

Executing the Scene Wizard

When the Scene Wizard is executed, it first closes the current scene description, and ask for saving if needed. If you had a valid scene description, the Scene Wizard compiles the scene to duplicate the original scene configuration.
