Clipboard and persistency

You can save a image in the Image Window into a file, with the option of making the background transparent. You can also copy the whole rendered image, or the current view of the image, to the Clipboard, to share it with other applications.

Save image

You have two commands, both in the toolbar and in the context menu of the Image Window, for saving the current image to a file:

Save image as always asks for the file name. Save image, on the contrary, only asks the first time the current image is saved. In both cases, the suggested file name is built from the name of the scene file, if any.

Copy to Clipboard

The Image Window supports three different commands for copying information into the Clipboard:

CommandShortcutAction
Copy imageCtrl+CCopies the whole rendered image to the Clipboard, ignoring zooming and any display options.
Copy viewCtrl+WCopies the current content of the Image Window, including zooming and any applied display options.
Copy position as textCtrl+Shift+CCopies the coordinates of the point below the cursor as text, in vector notation.
