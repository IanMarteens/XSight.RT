The Image Window

See also

The Image Window is automatically displayed when a scene is being rendered, and when the noise generator computes a bitmap. It shows the rendered image and it also allows some simple manipulations on the image.

In this section

Clipboard and persistency

You can save the rendered image into a file, with the option of a transparent background. You can also copy the whole rendered image, or the current view of the image, to the Clipboard, to share it with other applications.

Scene feedback

Once you have rendered a scene, you can extract more information from it using the rendered image and the internal scene tree, which is kept by the Image Window. For instance, you can zoom the rendered image in and out using the mouse wheel or keyboard shortcuts. You can also use the mouse and the status bar to find detailed information such as the distance from the camera to some point in the scene, the incidence angle, the normal at the surface of any visible object, or the color at the point below the mouse pointer.

Image source

Besides rendered images, the Image Window can display other images, such as the edge map of a scene. It also can store another bitmap, so you can display it later or use it to create a differential map.

Display options

The Image Window supports several easy non-destructive transformations on a rendered image and on its edge map. These transformations include color inversion, desaturation, sepia toning and gamma correction.
