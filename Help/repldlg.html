Replace Dialog Box

See also

The Replace dialog box allows searching the active document for a given string in order to replace it with new text. You can show this window by typing the Ctrl+R shorcut combination, or executing the Edit|Replace menu command, from the main menu.

Though it looks like a modal dialog, it is a floating tool window, so you can switch between this window and the code editor without closing the dialog first. You can turn this window into the Find dialog by type the Ctrl+F shorcut combination.
