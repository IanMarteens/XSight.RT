Multithreading

See also

The Multithreading check box in the left panel instructs XSight RT to use several concurrent threads for rendering scenes. It can reduce the required rendering time:

You must use this feature when rendering on a machine with two or more CPU cores. It can also save some time in old computers with only one hyperthreaded core. With traditional single core CPUs, dual mode will neither improve nor hurt performance.
