Find Dialog Box

See also

The Find dialog box allows searching the active document for a given string. You can show this window by clicking the Ctrl+F shorcut combination, or executing the Edit|Find menu command, from the main menu.

Though it looks like a modal dialog, it is a floating tool window, so you can switch between this window and the code editor without closing the dialog first. You can turn this window into the Replace dialog by type the Ctrl+R shorcut combination.
