The Scene Tree is a panel inside the editor window, showing which objects are present in a given compiled or rendered scene.

The scene tree

It can be activated both from the main menu, in the Editor Window, and from the context menu at the Image Window:

The source for the scene tree

In this case, the tree shown is the last compiled or rendered scene. You must realize that you can compile a scene without rendering it. You can have a scene compiled and rendered at the image window, and a different compiled scene at the editor window. The scene tree for an already rendered scene can be displayed using the context menu from the image window:

Using the scene tree

The main use for the scene tree is merely informative: you can check how the scene optimizer has modified the given scene.
