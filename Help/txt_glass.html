XSight RT

Materials: Glass

See also

This version of XSight RT includes experimental support for the glass material.

Please note the blend of reflected and refracted rays in the two glass spheres. The refracted image is inverted, but the reflected image, though distorted, is not. Observe how the intensity of the reflections depends on the angle of incidence.

Syntax

Glass materials are defined using these constructors:

  • Glass
  • Glass(ior)
  • Glass(ior, minReflection)
  • Glass(ior, minReflection, maxReflection)
  • Glass(ior, minReflection, maxReflection, phongAmount, phongSize)

The ior parameter represents the index of refraction. When not specified, an index of 1.5 is assumed: that's the index of refraction for glass. The default value for the maximum reflection is 1.0, and the minimum reflection, when omitted, is calculated from the ior using the Fresnel's formula:

minReflection = (ior - 1)^2 / (ior + 1)^2
